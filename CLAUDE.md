# SEO Content Map — Development Guide

## Stack
- **Framework**: Next.js 16 + React 19 (App Router)
- **Database**: SQLite via Prisma 7 + better-sqlite3 adapter
- **Styling**: Tailwind CSS 4, dark teal theme
- **AI**: Claude Sonnet 4 (via Anthropic API), Gemini Deep Research (brand intelligence)
- **Data**: DataForSEO API, Google Search Console API, PageSpeed Insights API
- **Export**: Google Sheets API

## Auth & Permissions
- Email/password auth with sessions stored in DB (no NextAuth)
- Session cookie: `seo_session`, 30-day expiry
- Middleware at `middleware.ts` redirects unauthenticated users to `/login`
- First visit to `/login` triggers admin setup flow (creates first admin user)
- Google OAuth (`lib/google-auth.ts`) is ONLY for GSC/Sheets API access, NOT for user login

### Roles (4 levels)
| Role | View | Edit/Checks | Run Diagnostic/Generate | Create Brands | Admin Panel |
|------|------|-------------|------------------------|---------------|-------------|
| viewer | Yes | No | No | No | No |
| editor | Yes | Yes | No | No | No |
| analyst | Yes | Yes | Yes | No | No |
| admin | Yes | Yes | Yes | Yes | Yes |

### Models
- `User` — email, passwordHash, name, role (admin/user), isActive
- `Session` — userId, token, expiresAt
- `BrandMember` — userId + brandId + role (viewer/editor/analyst). Admin has implicit full access
- `ActivityLog` — userId, brandId, action, details (JSON)

## Brand Sections (7 tabs per brand)
1. **Overview** (`/brands/[id]`) — KPIs, workflow checklist with completion checks
2. **Diagnostic** (`/brands/[id]/diagnostic`) — Rankings, SERP Analysis, Structure (sub-pages)
3. **Optimize** (`/brands/[id]/optimize`) — Quick Wins, Page Audit, Fixes, Cannibalization
4. **Content Map** (`/brands/[id]/content-map`) — Pillar/cluster strategy, Calendar, Briefs, Keyword Pool
5. **AEO** (`/brands/[id]/aoe`) — AI readiness scoring, per-page actions
6. **Speed** (`/brands/[id]/speed`) — PageSpeed/Core Web Vitals analysis, global + per-page issues
7. **Performance** (`/brands/[id]/performance`) — Weekly snapshots, trend charts, AI summary, impact tracking

## Key Architecture Decisions

### Target Keywords
- `brand.targetKeywords` — client-defined priority keywords (one per line in Settings)
- Used as priority seeds in keyword pool builder (before Claude-generated seeds)
- Tracked specifically in Performance "Impact of Our Work" section

### Performance Tracking
- `PerformanceSnapshot` model stores weekly snapshots (clicks, impressions, CTR, position, top10 count, keyword positions JSON)
- First visit backfills 12 weeks of history from GSC API
- Each subsequent visit saves this week's snapshot if not already saved
- AI summary generated by Claude comparing current vs previous period

### Content Map Flow
1. Keyword Pool Builder (GSC + DataForSEO related_keywords + Claude suggestions validated with DataForSEO)
2. Content Strategist (pre-groups by product stems, Claude selects pillars + clusters, 3/week schedule)
3. Content Brief (per-keyword, on demand, includes E-E-A-T and AEO optimization)
4. Quality Reviewer (ICP, cannibalization, funnel balance)

### PageSpeed
- Requires `PAGESPEED_API_KEY` in `.env.local` (free from Google Cloud Console)
- Analyzes top 20 pages by impressions (API takes 10-60s per page)
- CWV field data (CrUX) may show N/A for low-traffic sites — this is normal

## Development Notes

### Database changes
SQLite with better-sqlite3 adapter. `prisma db push` does NOT work with this config. To add columns:
```js
const Database = require('better-sqlite3')
const db = new Database('prisma/dev.db')
db.exec('ALTER TABLE TableName ADD COLUMN columnName TEXT')
```
Then update `schema.prisma` and run `npx prisma generate`.

### Claude JSON parsing
`lib/services/anthropic.ts` has robust JSON parsing: direct parse → code block extraction → brace-depth matching. System prompt appends "CRITICAL: Your entire response must be valid JSON."

### API caching
`ApiCache` model with TTL-based caching. PageSpeed cached for 3 days. Never cache empty SERP results.

### Environment variables
```
GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET, GOOGLE_REDIRECT_URI  # GSC/Sheets OAuth
DATAFORSEO_LOGIN, DATAFORSEO_PASSWORD
ANTHROPIC_API_KEY
GEMINI_API_KEY
PAGESPEED_API_KEY
```
All in `.env.local` (gitignored). Never commit secrets.
